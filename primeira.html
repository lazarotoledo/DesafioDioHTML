Estrutura e Tags

A estrutura básica do HTML para inciantes é composta pelas tags mostrada na imagem acima.

Estrutura Básica:

As tags de abertura não contém o /, somente a de fechamento. Nem todas tags tem fechamento.
  1. A tag HTML limite o início e o fim do código.

    2. <.HTML> <./HTML>*

  2. Dentro da tag HTML colocando o cabeçalho e o corpo do código.

    <.head><./head>*

    O Cabeçalho do HTML.

    <.body><./body>*

    O corpo do HTML, onde ficará o que vai ser visto pelo usuário.

TAGS

Podemos citar algumas tags interessantes para o aprendizado

Tags de para título de texto:

A família H, que vai da H1 até a H6

Abertura<.h número> fechamento<./h número>

  1. H1: O principal,maior caixa

  2. H2: início do subtítulos

  3. H3: Diminui

  4. H4: Vai diminuindo

  5. H5: diminuindo
  6. H6: Até chegar no menor.

Parágrafo

Define o começo e o fim de um parágrafo.

Podemos também ser alinhado de acordo com a necessidade, na esquerda com padrão, meio ou direita.

Quebra de Linha

Tags utilizadas para quebra de linha em um texto. É mais comum a utilização da tag fechada <./br>

<.br><./br>

TAG de Style

Essa tag define o estilo de parte ou de todo o conteúdo da página web com arquivo .css importado.

<.style><./style>
