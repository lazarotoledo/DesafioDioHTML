TAG SUB

É usada para adicionar um texto subscrito ao documento HTML, deixando o texto selecionado entre a tag SUB na parte debaixo da linha. O texto subscrito pode ser usado para fórmulas químicas, como H2O para ser escrito como H2 O. Ficando abaixo da linha do texto.

<.sub> Esse texto subscrito<./sub>

TAG SUP

Sendo o oposto da tag anterior, deixando o texto selecionado entre a tag SUP na parte de cima da linha

<.sup>Esse texto Sobrescrito<./sup>

TAG DEL

Essa tag é utilizado para definir o texto que foi excluído de um documento. Geralmente os navegadores atingem uma linha através do texto, ou seja, palavra cortada.

<.del> <./del>

Esse TEXTO foi cortado

É interessante colocar um background para destacar a palavra ou texto dentro da tag.

Para fazer o background, pode ser feito definindo no head um tag style de del com background ou dentro da tag del que deseja destacar.

Links

Links HTML são hiperlinks.
Pode clicar em um link e ir para outro documento.

<.a href="url">texto links<./a>

  • O atributo mais importante do elemento é o atributo href que indica o destino do link.<.a>
  • O texto do link é a parte que ficará visível para o leitor.

Quando você move o mouse sobre um link, a seta do mouse se transforma em uma mãozinha.

IMAGEM

  • A tag é usada para incorporar uma imagem em uma página HTML.<.img>
  • As imagens não são tecnicamente inseridas em uma página web; Imagens estão vinculados a páginas da web.
  • A tag cria um espaço de retenção para a imagem referenciada.<.img>

<.img src="img_girl.jpg" alt="Girl in a jacket" width="500" height="600">

A tag tem dois atributos obrigatórios:<.img>
>src - Especifica o caminho para a imagem
>alt - Especifica um texto alternativo para a imagem, se a imagem for para alguns o motivo não pode ser exibido
Nota: Além disso, sempre especifique a largura e a altura de uma imagem.
Se a largura e a altura não forem especificadas, a página poderá piscar enquanto a imagem Cargas.
